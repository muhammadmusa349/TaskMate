# 📝 TaskMate

A beautifully designed, premium task management mobile application built with **Flutter** and **Firebase**. 

TaskMate is built with a focus on seamless user experience, responsive design, and real-time cloud synchronization. Whether you are managing daily chores or long-term projects, TaskMate keeps your goals organized with an intuitive, fluid interface that adapts to your device's theme.

---

## ✨ Key Features

* **🔐 Secure Authentication:** Complete user flow including Sign Up, Login, and a secure Forgot Password reset mechanism, all powered by Firebase Authentication.
* **☁️ Real-Time Cloud Sync:** Tasks are instantly saved, updated, and synced across devices using Firebase Cloud Firestore.
* **🌗 Adaptive Dark & Light Mode:** A dynamic UI that automatically responds to your system's theme settings, complete with custom borders and plush shadows for optimal visibility day or night.
* **⚡ Fluid Interactions:** Enjoy premium UX touches like swipe-to-delete functionality and smooth, animated bottom-sheet forms for task creation and editing.
* **🎯 Smart Organization:** Structure your day by setting task priorities (Low, Medium, High) and assigning calendar due dates. Filter views easily between pending and completed tasks.
* **📊 Dynamic User Dashboard:** A personalized profile screen featuring a real-time statistics board that tracks your productivity and task completion rates.

---

## 🛠️ Tech Stack

* **Frontend:** [Flutter](https://flutter.dev/) (Dart)
* **Backend:** [Firebase](https://firebase.google.com/)
* **Database:** Cloud Firestore (NoSQL)
* **Authentication:** Firebase Auth
* **State Management:** Provider pattern
